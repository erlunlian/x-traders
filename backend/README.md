Environment variables for LLM rate limiting (optional):

- LLM_AZURE_MAX_CONCURRENT (default 2)
- LLM_AZURE_RPS (default 0.5)
- LLM_AZURE_BURST (default 2)
- LLM_OPENAI_MAX_CONCURRENT (default 4)
- LLM_OPENAI_RPS (default 1.0)
- LLM_OPENAI_BURST (default 4)
- LLM_ANTHROPIC_MAX_CONCURRENT (default 3)
- LLM_ANTHROPIC_RPS (default 0.8)
- LLM_ANTHROPIC_BURST (default 3)
- LLM_XAI_MAX_CONCURRENT (default 2)
- LLM_XAI_RPS (default 0.5)
- LLM_XAI_BURST (default 2)
- LLM_MAX_RETRIES (default 5)
- LLM_RETRY_BASE_DELAY (default 1.0)
- LLM_RETRY_MAX_DELAY (default 15.0)
